# Chrono Class 
**Sistema de Gestión Académica Universitaria** desarrollado en C++.

## Descripción
Chrono Class permite organizar el horario semanal (7 bloques x 6 días) de una institución educativa utilizando estructuras de datos avanzadas.

## Características Técnicas
* **Memoria Dinámica:** Implementación de matrices mediante punteros dobles (`string**`).
* **Gestión por Referencia:** Uso de punteros para el registro directo de cursos en memoria.
* **Reportes:** Visualización completa del horario y filtrado por días específicos.
